# Other Magic-Related Commands in Pathfinder Second Edition

`spell <spell name>`: Looks up the details of a spell. It will accept partial matches, but it will display details only if there is only one match.
`spellbook`: Displays all spells known by your character.
`magic [<character name>]`: Displays details of spell availability by class for `<character name>`, or yourself if name is not given. Displays all casting progressions and focus spell lists. **NOTE:** This command does not show what you have prepared (if you are a prepared spellcaster), only what you have readied - i.e. what you can actually cast before refreshing. See `help prepare` for more information.

The `spell/search` command provides a robust search function to help you find spells for your character to learn. See `help spell search` for more information.
